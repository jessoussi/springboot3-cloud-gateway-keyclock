# springboot3-cloud-gateway-keyclock
Integration of Spring Cloud Gateway with Keylock with Openid Connect protocol


1- Run Keyclock

$> cd $KEYCLOCK_HOME/bin
$> kc.sh start


2- Create Realm in keyclock

go to http://localhost:8080
